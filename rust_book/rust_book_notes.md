# 1. Getting Started
## Install
* Install Rust for windows using [this](https://www.rust-lang.org/tools/install)
* Check version to ensure proper install 
    ```
    rustc --version
    ```

## Hello World
* Rust files have a `.rs` extension
* Use underscores to separate words in script file names
    * e.g.- `hello_world.rs`
* Running scripts
    ```
    rustc main.rs
    ```
